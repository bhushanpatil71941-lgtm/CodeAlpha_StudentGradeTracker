# Student Grade Tracker

A desktop-based Java Swing GUI application designed to track and manage student grades.

## Key Features

1. *Real-Time Analytics Dashboard*
   Instantly updates class average, highest score, lowest score, and student count.

2. *Automated Grading System*
   Automatically calculates student grades (O, A+, B, etc.) and PASS/FAIL status based on marks.

3. *Live Search Filter*
   Allows quick searching and filtering of student records dynamically as you type.

4. *Database Table Directory*
   A clean, structured JTable layout to view, manage, and delete student records easily.

5. *Print & PDF Export*
   Integrated system print dialog to instantly print the student records or save them as a PDF.
